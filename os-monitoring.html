Description

Revolance OS Suite is a set of tools to view changes on the OS FileSystem (file updates, file creation), OS Processes (creation, Memory consumption, CPU consumption) in order to know client side action impacts on the server side. This simplify testcases diagnostics in case of failure.

Features

Java process monitoring

Real time charting

Monitoring and correlation of several Jvms

Components

a library

a server

a memory consumer sample

Roadmap

non java processes monitoring

heap dump generation

heap dump viewer/analyzer
