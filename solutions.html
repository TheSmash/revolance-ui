Description

Revolance UI Suite is an alternative to declarative UI testing. UI tests requires a lot of maintenance, which can rapidly grow into a nightmare. Revolance UI analyse your running application UI. All pages are automatically analysed: buttons, links, Images... and even more! Then you can review the changes of your application. It's as you do it with a classic SCM client except that for an HTML UI you need to track positions, looks, content, ...

Features

100% HTML5/CSS3 compatible

latest FF & chrome support

configure the browser screen size

login pages support

regression detection

multiple tab support

Components

an explorer: explores & generates reports

a comparator: compares reports

a parser: retrieve page details

a database: stores the reports

a server: provide APIs and a UI

The Change Review board

The left page is the reference, on the middle is the reference you're building
and the right one is the current page

Automatic review

The content that has not been changed (look, position, content, size, target url)
is automatically taken care of (in the centered page).

Highlight of changed content

If an element disappeared from the reference, then it's marked in red
If an element appeared in the new reference, then it's marked in gold
You'll have to merge it manually (review process) by dragging and dropping elements to the middle page
